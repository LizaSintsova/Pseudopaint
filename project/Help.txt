Для начала работы программы установите python, pymorphy2, PyQt5
Установка python: https://www.python.org/downloads/
Для установки pymorphy2, PyQt5 откройте cmd и ввидите туда: 
pip install pymorphy2
pip3 install pyqt5
После чего запустите Pseudopaint

Интерфейс программы:
Основные цвета - удобные часто нужные цвета в работе 
Выбор цвета - выбор нужного цвета
Кисть - рисование кистью
Квадрат - рисование черными квадратами
Сохранить - сохранение рисунка в двух возможных форматах (JPNG, PNG)
Цвет заливки - цвет фона
Размер кисти - измение размера кисти и квадрата 
Что нарисовать - генератор идей для рисунка
